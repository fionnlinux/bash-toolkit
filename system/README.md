# system

Scripts for monitoring and reporting on system status.

## Scripts

### system-info.sh
Displays basic system information in a readable format.

**Output includes:**
- Hostname
- Uptime
- Kernel version
- Disk usage (root filesystem)
- Free memory

**Usage:**
\```bash
./system-info.sh
\```

**Example output:**
\```
Hostname:    fedora
Uptime:      up 58 minutes
Kernel:      7.0.12-201.fc44.x86_64
Disk usage:  2%
Memory free: 20Gi
\```
